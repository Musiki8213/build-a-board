POWER UNIT

The power supply slot provides electricity to start and run the system.
It uses a 20-pin connector to deliver about 110V AC power.

CPU

×

CPU (Central Processing Unit)

The CPU handles all data processing, calculations, and output formatting.
It works in cycles (MHz), like a heart beating e.g., 2000MHz.

RAM

×

RAM (Random Access Memory)

RAM is temporary memory for processing active data.
It allows fast, random access to information.

ROM

×

ROM (Read-Only Memory)

ROM stores startup instructions and stays intact when power is off.
It cannot be changed by the user.

CMOS

×

CMOS (Complementary Metal-Oxide-Semiconductor)

CMOS stores BIOS settings and real-time clock data using a battery.

STORAGE DRIVE

×

HDD (Hard Disk Drive)

HDD stores all files and system data. It can read and write data.
Internally, it has spinning disks like pancakes.
